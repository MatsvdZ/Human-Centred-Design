# HCD
Mijn repo voor het vak HCD voor de minor Web design and development

# Welk testpersoon heb ik?
Ik heb voor deze opdracht Darice als testpersoon gekregen. Voor Darice kunnen we kiezen uit twee opdrachten. Dat zijn de volgende:

1. Darice wil graag de inhoud van podcasts kunnen volgen, inclusief alle hoorbare details zoals sfeergeluiden, geluidseffecten, en nuances zoals intonatie en andere eigenschappen van de stemmen. Is de persoon bijvoorbeeld boos? Verdrietig? Vrolijk?

2. Ze wil films kunnen volgen, ook de hoorbare details zoals geluidseffecten, (spannende) muziek, etc.

Ik heb gekozen voor de film opdracht. Ik denk dat ik hier gewoon net wat meer kan doen dan bij alleen een podcast, en ik kan hier denk ik ook wat meer leuke dingen mee doen die de ervaring zeker zullen bevorderen.

Om te beginnen heb ik meteen even gekeken of ik Darice online kon vinden, en dat was al snel gelukt. Ik heb een blog gelezen over diversiteit onder dove mensen. Hierin staat onderaan dat Darice een voorkeur heeft voor ondertiteling, hier kan ik dus zeker mee aan de slag.

# Wat zijn de mogelijke problemen (aanname)?
Gewone ondertiteling laat vaak alleen normale gesproken tekst zien. Hierdoor kan je belangrijke hoorbare informatie missen, zoals toon, spanning, muziek en geluidseffecten. Dit wil ik aanpakken

## Welke funties zijn hier voor nodig?
- Videospeler
- Ondertiteling (die live meeloopt met de gesproken tekst?)
- Labels/kleuren voor emoties
- eventueel aparte regels voor geluidseffecten en muziek?
- Visuele effecten bij spanning of harde geluiden

## Wat zou mijn ontwerp nodig hebben?
- Aanpasbare letter grootte
- Contrast?
- Waar moet de ondertiteling?

Ik kan bijvoorbeeld het volgende doen: 
- Gesproken tekst: wit
- geluidseffecten: labels in de tekst?
- Muziek/sfeer: ook labels?
- emotie: spelen met size en animaties in de tekst?
